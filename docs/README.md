| Document | Feature | Status |

| `GOOGLE_SHEETS_BACKEND_PLAN.md` | Google Sheets integration | Deferred |
| `SUPABASE_HEARTBEAT.md` | Supabase heartbeat workflow | Shipped |
| `SUPABASE_MIGRATION_PLAN.md` | Supabase backend migration | Shipped |
| `YEAR_TRACKING_PLAN.md` | Multi-year data tracking | Shipped |

For the current system architecture, see `SMOKE_TEST.md` for the deployment checklist and `src/lib/grme-data.ts` for the core data model.
